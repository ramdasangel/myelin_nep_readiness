SRI Framework Overview

The School Readiness Index (SRI) measures a school branch's readiness to implement India's National Education Policy (NEP 2020) across five constructs. The maximum total score is 100 (85 achievable with current data).

C1 Intent (20)
C2 Practice (25)
C3 Capacity (20)
C4 System (20)
C5 Ecosystem (15)

C1 — Intent Readiness (max 20)

Data Source

Teacher intent surveys (setCodes 1234/103) and leader intent surveys (setCodes 7890/104). 22 questions for teachers, 20 for leaders. ~15,500 response rows across English and Marathi.

Goal → Focus Point Mapping

  • FP1 — Each Child is Unique (Teacher) / Understanding Learners (Leader EN)
  • FP2 — Competency-Based Learning (Teacher) / Classroom Quality (Leader EN)
  • FP3 — Teacher Upskilling (Teacher) / Professional Development (Leader EN)
  • FP4 — Diagnosing Learning Levels (Teacher) / Data Use (Leader EN)
  • FP5 — Parent-Teacher Collaboration (both roles)

Leader Marathi surveys use the same 5 goals as teachers (translated).

Computation

ADV[fp] = mean(selectedOption for questions in fp) / 3.0 // Aspirational Depth Vector, normalized to [0,1]
Coverage = count(ADV[fp] > 0.10) / 5 // fraction of FPs above threshold τ=0.10
Balance = max(0, 1 − Var(ADV) / 0.16) // 0.16 = max variance for 5 uniform bins
Alignment = cosine_similarity(teacher_ADV, leader_ADV)

TeacherScore = TeacherCoverage × TeacherBalance × Alignment
LeaderScore = LeaderCoverage × LeaderBalance × Alignment
C1 = 10 × TeacherScore + 10 × LeaderScore

Note: Branches with only one role (teacher or leader) get Alignment=0, resulting in C1=0. This is by design — intent readiness requires both roles to be aligned.

C2 — Practice Readiness (max 25)

Data Source

Same teacher intent responses (setCodes 1234/103). selectedOption 0-3 mapped to depth levels D1-D4.

Computation

PDDM[fp][depth] = count of teacher responses at each depth // 5×4 Practice Depth Distribution Matrix

Non-observable cells masked to 0:
  FP1-D1 (opt 0), FP1-D4 (opt 3), FP2-D4 (opt 3)

Achievable max depth after masking:
  FP1: 2.0, FP2: 2.0, FP3-FP5: 3.0

E_depth[fp] = weighted_avg(depth × masked_count) / total
E_norm[fp] = E_depth[fp] / max_depth[fp]
CoverageFP = covered_FPs / 5
DepthEffectiveness = CoverageFP × mean(E_norm)
C2 = 25 × DepthEffectiveness

C3 — Capacity Readiness (max 20)

Data Source

MathTangle adaptive cognitive assessment (mathangle_master.csv). 107 respondents across 28 branches. Measures cognitive levels (Understand/Apply/Analyze/Evaluate), difficulty levels (L1-L3), and behavioral metacognition.

Computation — 4 CPD Dimensions

CPD1 (Higher-order) = (Cog:Analyze_Right + Cog:Evaluate_Right) / all_cog_total
CPD2 (Decomposition) = (L2_accuracy + L3_accuracy) / 2
CPD3 (Revision) = 1 − IndecisionScore / max_indecision_across_all_users
CPD4 (Consistency) = 1 − FlipFlops / TotalQuestions

CPI_user = mean(CPD1, CPD2, CPD3, CPD4)
CPI_branch = mean(CPI across all users in branch)
C3 = 20 × CPI_branch

C4 — System Readiness (max 20)

Data Source

Stage-2 baseline survey (system_readiness_responses_detail.csv). 12,576 response rows. 68 questions across English and Marathi. Teacher areas: A1 (Continuous Learning Diagnostics), A2 (Teacher Development), A3 (Holistic Progress Card), A4 (Parent & Community). Leader areas: B1 (NEP Governance), B2 (Data-Informed Decisions), B3 (Teacher Development Culture), B4 (HPC & Reporting), B5 (Parent & Community Partnerships).

Computation

Binary = 1 if selectedOption ≤ 1 (SA/A), 0 if selectedOption ≥ 2 (D/SD)

A_mean[area] = mean(binary) for each area A1..A4
B_mean[area] = mean(binary) for each area B1..B5

EnablementIndex = mean(A1_mean, A2_mean, A3_mean, A4_mean)
RoutineIndex = mean(B1_mean, B2_mean, B3_mean, B4_mean, B5_mean)
C4 = 10 × EnablementIndex + 10 × RoutineIndex

Note: Branches without leader respondents get RoutineIndex=0, capping C4 at ~10.
